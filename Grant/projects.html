Project Kepler

Burp Suite Extension

Project Kepler is designed to help penetration testers catalog their attacks and document manual testing efforts. It allows you to "record" specific requests, categorize them (e.g., SQLi, XSS), assign a status (Vulnerable, Safe), and add notes. This prevents redundant testing and facilitates knowledge sharing within teams.

View on GitHub

I have also developed components for custom enterprise extensions for Burp Suite using the Java-based Montoya API to automate specific testing workflows in professional environments.

Self-Hosted Docker Homelab

A practical application of DevSecOps and automation.

This website itself is a project, running on the Bun runtime environment within a Docker container on a home server. The entire homelab environment runs containerized services (including n8n for AI automation) and is securely exposed to the internet via secure tunneling. This setup gives me a hands-on lab for experimenting with network security, containerization, and secure infrastructure management.
